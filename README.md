# generator-antd-custom 脚手架

#### yeoman + download-git-repo + fs-extra + latest-version


---

## 说明

使用yeoman框架，实现ant-custom项目脚手架，同时校验脚手架的最新版本并更新

## 安装

```bash
$ npm i yo -g
$ npm i generator-antd-custom -g

```

## 初始化项目
```bash
$ yo antd-custom
```

## 运行项目

```bash
$ npm start
```
## 打包项目

```bash
$ npm run build:test
```
or
```bash
$ npm run build:prod
```
