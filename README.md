# generator-antd-custom 脚手架

---

## 说明

对之前的版本升级改造，实现自定义命令行，配合antd-custom-tpl模板运行

## 分支
master 稳定版本

develop 开发版本

test yeoman命令版本 


## 安装

```bash
$ npm i generator-antd-custom -g

```

## 初始化项目
```bash
$ cfe init react
```

## 运行项目

```bash
$ npm start
```
## 打包项目

```bash
$ npm run build:test
```
or
```bash
$ npm run build:prod
```
